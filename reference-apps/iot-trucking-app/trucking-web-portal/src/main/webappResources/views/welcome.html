Welcome to the HDP 2.5 Streaming Analytics Reference Application

The specific components in this architecture are the following:

  • Storm Stream Processing Framework to process real-time streams of trucking events
  • HBase Store the Time-Series Trucking Events for Fast Access
  • HDFS Store All Trucking Events in HDFS for Analytics
  • Solr Index trucking events for Discovery and Analysis
