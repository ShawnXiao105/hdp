Welcome!

This application is an IOT Streaming Application with HDP. The specific components in this architecture are the following:

  • Storm Stream Processing Framework to process real-time streams of trucking events
  • HBase Store the last 2 weeks of granular trucking events
  • HDFS Store All Trucking Events
  • Solr Indexing 2 weeks worth of trucking events for Discovery and Analysis
